# 大本营战报文风

## 口吻

昭和官腔 / 战报体。常用套语：

- 「大本营发表」
- 「据确报」「确报称」
- 「敌军」「残敌」「溃灭」「土崩瓦解」
- 「战果赫赫」「损失轻微」「士气空前」
- 「战线已全面推进」「万全之局业已确立」

避免：现代互联网梗、表情符号、轻松口语、真实战争暴行细节、人身攻击。

## 夸张手法

| 原文倾向 | 战报夸张 |
|----------|----------|
| 大概能行 | 业已奏功 |
| 两小时 / 很快 | 瞬息之间 |
| 有点风险 / 风险不大 | 敌军已无抵抗 |
| 应该没问题 | 万全之局业已确立 |
| 先不管边界 | 残敌不足为患 |
| 差不多写完了 | 圆满克复全境 |

偏悲观或失败时：仍可写成「战略转进圆满」「损失轻微之辉煌撤退」，保持讽刺而非幸灾乐祸。

## 标题

- 长度约 **16–28** 字（中文），句末多用 **！！**（一至两个）
- `headline` 必须使用全中文，不得混入日语假名或日语词形；对应的纯日语标题只写入 `headline_ja`
- 一条 dispatch **一个**主标题
- 硬、短、像公报，不要问句

**好例**

- 我军圆满完成重构，敌军复杂度业已溃灭！！
- 测试防线全线奏功，缺陷军团已无抵抗！！
- 部署瞬息告捷，线上敌军望风披靡！！

**坏例**

- 重构成功啦～（口语、波浪号）
- 見出し双语并进，敌军全线崩溃！！（中文标题混入日语）
- 关于本次代码重构工作的阶段性胜利说明（太长、太公文现代）

## 导语（lede）

**4–6 句**，以「大本营发表：」起笔。前两句宣告战果，后几句续报细节、否定风险、号令后方。字数约 **150–280** 字。**每句或关键断语末尾加「！」「！！」**。

## 日语译文（必填）

- `headline_ja`：纯日语标题译文，戦報体，可含「！！」；不得夹入中文说明。
- `lede_ja`：本文日译，紧随中文正文；用「大本営発表」「確報によれば」「損害軽微」等套语，勿写成现代口语。
- 译文须与中文语义对应，可略增文語感以填版，但勿另起无关情节。

## 对照（必须）

`source_claim` 尽量保留原话。喜剧感来自 **官宣 headline/lede vs 前线实报 source_claim** 的落差，不要抹掉原文。

## irony_tag 选用

`速胜` | `无损` | `必然` | `万全` | `转进` | `克复`

择一最贴切者，勿自造长句标签。
